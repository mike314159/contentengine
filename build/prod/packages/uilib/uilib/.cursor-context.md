# UILib Flask App Context

## Flask App (`app.py`)

A simple Flask application with URL testing capabilities.

### Features
- **Single route**: `GET /` returns "hello"
- **URL testing**: Test any route without starting the server
- **Debug mode**: Enabled for development
- **Port**: Runs on 5001 (to avoid conflicts)

### Usage

#### Test a single URL (recommended for development)
```bash
python app.py --url "/"
python app.py --url "/nonexistent"
```
- Tests the specified URL and returns full response details
- Exits immediately after testing
- Shows status code, content type, content length, and full response

#### Run the development server
```bash
python app.py
```
- Starts Flask development server on http://127.0.0.1:5001
- Auto-reloads on file changes
- Debug mode enabled

### Command Line Arguments
- `--url URL`: Test a single URL and exit (e.g., `--url "/"`)
- `--help`: Show help message
